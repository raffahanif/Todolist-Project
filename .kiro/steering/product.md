# Product

A client-side Single Page Application (SPA) for managing todo tasks. Users can create, read, update, and delete todos, mark them as complete, and filter by status. All data is persisted in the browser's localStorage — no backend required.

## Core Features
- Add, edit, delete todos
- Toggle completion status
- Filter by: All, Active, Completed
- Bulk delete all completed todos
- Summary count of completed vs total todos
- Data persists across page reloads via localStorage

## Target Users
End users managing personal task lists via a web browser.
